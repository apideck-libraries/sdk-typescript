# ExternalDetails

Details about an external payment.

## Example Usage

```typescript
import { ExternalDetails } from "apideck/models/components";

let value: ExternalDetails = {
  type: "social",
  source: "<value>",
  sourceFeeAmount: 2.5,
};
```
