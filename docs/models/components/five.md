# Five


## Supported Types

### `string`

```typescript
const value: string = "Uses Salesforce and Marketo";
```

### `number`

```typescript
const value: number = 10;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `{ [k: string]: any }`

```typescript
const value: { [k: string]: any } = {
  "foo": "bar",
};
```
