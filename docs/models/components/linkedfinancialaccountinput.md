# LinkedFinancialAccountInput

A flexible account reference that can represent either a ledger account (GL account) or a bank account, depending on the connector's requirements.

## Example Usage

```typescript
import { LinkedFinancialAccountInput } from "@apideck/unify/models/components";

let value: LinkedFinancialAccountInput = {
  id: "123456",
  type: "ledger_account",
  code: "1100",
  displayId: "123456",
  accountNumber: "123465",
};
```
